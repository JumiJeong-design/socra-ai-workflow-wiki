# Screen Design Playbook

## When To Use

Socra 제품의 새 화면이나 주요 화면 개편을 AI와 함께 설계할 때 사용한다.

## Steps

1. 화면의 핵심 사용자 목적을 한 문장으로 적는다.
2. 필요한 기존 design system component를 먼저 고른다.
3. Figma에서 화면 초안을 만들고 component gap을 표시한다.
4. gap이 반복 가능한 컴포넌트라면 `socra-ai-product-design`에 component spec 초안을 만든다.
5. AI design review checklist로 사용성, 상태, 반응형, 접근성을 점검한다.
6. 결정된 패턴은 workflow guide나 playbook으로 승격할지 판단한다.

## Done Criteria

- 화면이 디자인 시스템 컴포넌트 기반으로 설명된다.
- 새 컴포넌트 후보와 one-off UI가 구분된다.
- open question과 designer judgment가 분리되어 있다.
