04

이름을 유지하는 문서

도구가 자동으로 찾는 파일은 번호를 붙이지 않는다. README.md, AGENTS.md, CLAUDE.md는 root entry라 파일명을 유지한다. skills/**도 discovery 경로와 연결될 수 있어 rename하지 않는다.

05

git 문서 검토 순서

  1. 먼저 이 페이지에서 번호 의미를 확인한다.
  2. repo의 README.md 또는 AGENTS.md로 들어간다.
  3. 현재 상태는 10, 규칙은 20, 절차는 30, 검증은 40, 계약은 50 문서에서 찾는다.
  4. 70이나 90 문서는 원본 기록/과거 참고로만 보고, 현재 기준으로 바로 실행하지 않는다.
